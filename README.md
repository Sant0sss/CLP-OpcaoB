## 🖥️ Compilação e Execução  

### 💻 No Windows:  
Para compilar:  
```bash
gcc gauss.c -o gauss.exe -lm
````
Para executar:
```bash
gauss
````
### 🐧 No Linux/macOS:
Para executar:
```bash
./gauss
````
###🦀 Em Rust:
Para compilar:
```bash
rustc gauss.rs -o gauss
````
Para executar:
```bash
./gauss
````
### 🐹 Em Go:
Para executar diretamente:
```bash
go run gauss.go
